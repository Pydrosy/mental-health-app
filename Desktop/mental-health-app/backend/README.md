# Mental Health Web Application Backend

## Overview
Backend API for a mental health web application that connects patients with therapists using Graph Neural Networks for personalized matching.

## Features
- 🔐 JWT Authentication & Authorization
- 👥 User Management (Patients & Therapists)
- 💬 Real-time Chat (Socket.io)
- 📹 Video Call Integration (Agora)
- 📅 Session Scheduling
- ⭐ Rating & Review System
- 📊 GNN-based Therapist Matching
- 📧 Email Notifications
- 📁 File Uploads

## Tech Stack
- Node.js & Express
- MongoDB with Mongoose
- Socket.io for real-time communication
- JWT for authentication
- Agora for video calls
- Nodemailer for emails
- Multer for file uploads
